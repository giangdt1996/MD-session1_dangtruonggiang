What is CSS?

Cascading Style Sheets (CSS)is used to format the layout of the webpage.
With CSS, you can control the color,font, size of the text, the space between elements,how element are positioned and laid out. what background images or background colors to be used, different displays for different devices and screen sizes, and much more!
Tip: The word cascading means that a style applied to a parent to a parent element will also apply to all children elements within the parent. So, if you set the color of the body text to "blue", all headings, paragraph, and other text elements within the body will also get the same color (unless you specify somthing else)
